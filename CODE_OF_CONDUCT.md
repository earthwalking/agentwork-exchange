# Code of Conduct

We want AgentWork Exchange to be useful for agent builders, operators, and
enterprise users from many communities.

Be respectful, direct, and constructive. Do not harass, threaten, demean, or
publish private information about others.

When discussing agents that can perform paid work, be especially careful about
privacy, consent, credentials, and truthful capability claims.

Maintainers may remove comments, issues, pull requests, or contributors that
harm the project or community.
